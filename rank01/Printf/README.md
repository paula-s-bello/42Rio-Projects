# Description

This project is a reimplementation of the printf function from the standard library of C.
Hes part of the 42 Rio curriculum, and has the purpose to get deep in the usufull ways of how to use:
variadic variables, formatting and string manipulation.

- Printing characters (%c)
- Printing strings (%s)
- Printing integers (%d and %i)
- Printing unsigned numbers (%u)
- Printing hexadecimal numbers (%x and %X)
- Printing pointers (%p)
- Printing the percent sign (%%)
